Trilith Project : Yet Another Oauth2 ATM
########################################

Trilith is a project providing a unified way to generate,
keep track of and manage OAuth tickets.

Based on WSGI and `OAuthlib`, it aims at providing a clear way
to implement a OAuth Provider without giving up the pluggability and
customisation.

No microframework heren just pure Python.

The example UI ``trilith.admin`` is implemented using the Cromlech framework.
